### 머쓱이보다 키 큰 사람

```java
import java.util.*;

class Solution {
    public int solution(int[] array, int height) {
        int answer = 0;
        
        for(int i=0; i<array.length; i++){
            if(height<array[i]){
                answer++;
            }
        }
        return answer;
    }
}
```
1. 인자로 받은 height값이 array[i] 즉, 배열 값 안에 든 값보다 높다면 answer을 ++ 해줌
